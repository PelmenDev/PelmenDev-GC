PelmenDev GC for CS:GO
About
This project contains the PelmenDev GC.pbin file for CS:GO that works only with Csgo_Gc. Please note that this project is developed and maintained by a single person working in their free time. The developer is not experienced, so please be patient and understanding.

For more information about Csgo_Gc, see the original repository:
https://github.com/mikkokko/csgo_gc

Installation Guide
To install the PelmenDev GC.pbin file, follow these steps:

Locate your CS:GO installation folder. By default, it should be:
Steam\steamapps\common\csgo legacy\csgo\panorama
Copy the PelmenDev GC.pbin file into the panorama folder.
Launch CS:GO and ensure that Csgo_Gc is running, as this Code.pbin works only with it.
